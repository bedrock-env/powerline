### Setup

Clone this project to ``[BEDROCK_DIR]/extensions```. For example:

```sh
cd ~/Development/bedrock/extensions
git clone git@github.com:bedrock-env/powerline.git
```

```sh
cd ~/Development/bedrock
./scripts/bootstrap
```
